# Журнал проекту

Тут фіксуються емпіричні пороги, рішення й спостереження — що працює, а що ні. Ведеться з першого дня, щоб згодом на реальних даних вибудувати логіку керування.

## Агрономічні пороги

Огірки люблять тепло й високу вологість — приблизно 24–28°C і 75–90% вологості повітря. Бояться протягів. Пріоритет для них — полив і тепло. Сигналити треба, коли стає холодно або сухо.

Помідори навпаки — для них критично низька вологість, орієнтовно 60–70%. Якщо вологість зависока, з'являється ризик фітофтори й гнилі. Пріоритет — вентиляція й провітрювання. Сигналити треба, коли вологість піднімається вище норми.

Це стартові орієнтири з опису проекту. Реальні пороги уточнюватимуться за сезон на фактичних вимірах.

## Рішення по залізу

868 МГц — легально правильна LoRa-частота для України та Європи. Це не обговорюється.

Ємнісні датчики вологості ґрунту версії 1.2 обрано замість резистивних: резистивні кородують і ненадійні. Абсолютна точність тут не потрібна — достатньо для тригера поливу. Калібрування гравіметричне, кухонними вагами й духовкою.

SHT31 обрано замість DHT11/DHT22 через комерційну надійність для реального розгортання.

Heltec V3 обрано замість V4 — платформа зріліша, а конектор для сонячної панелі у Фазі 1 не потрібен.

Raspberry Pi 5 обрано замість вживаного міні-ПК — офіційний комплект прибрав різницю в ціні. microSD SanDisk/Kingston з рознесенням зносу через цілодобовий запис у базу.
